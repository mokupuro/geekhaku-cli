## Installation

- go

```
$ go get github.com/mokupuro/geekhaku-cli
```

- brew

```
$ brew tap mokupuro/geekhaku-cli
or
$ brew install mokupuro/geekhaku-cli/geekhaku-cli
```

## Update

```
$ brew upgrade mokupuro/geekhaku-cli/geekhaku-cli
```
